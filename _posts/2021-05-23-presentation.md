점화식을 잘못 생각해내어 틀렸었다.
2X2 타일마다 3가지 경우의 수가 있으므로 칸이 두칸 늘어날때마다 3배씩 늘어난다 생각했었다.

### 문제 설명

주어진 N에 대하여 2XN 크기의 바닥을 1X2,2X1,2X2의 타일로 채우는 경우의 수를 출력하면 된다.

# 입력 조건

- 첫 번째 줄에 N이 입력된다. (1<=N<=1,000)

# 출력 조건

- 경우의 수를 796,796으로 나눈 나머지를 출력한다.

### 답안 코드

```
#include <cstdio>

int main(){
    int n;
    scanf("%d",&n);
    int dp[n+1];
    dp[1]=1;
    dp[2]=3;
    for(int i=3;i<=n;i++){
        dp[i] = (dp[i-1]+2*dp[i-2])&796796;
    }
    printf("%d",dp[n]);
    return 0;
}
```

### 코드 해설

1. 라이브러리
   <cstdio> 기본 입력을 위한 라이브러리이다.
2. main
   우선 2X2까지는 특수한 경우이므로 초깃값을 설정해주고, 그 후로 세운 점화식에 따라 bottom-up 방식으로 ~~메모이제이션~~을 한다. 여기서 점화식은 가로길이가 i라고 할 떄 i-1번째와 i-2번째의 경우의 수를 고려해주면 되는데, 그 이유는 i-1번째에는 경우의 수가 한가지 밖에 없고 i-2번째에는 경우의 수가 2가지이다.(3가지로 착각했던 것은 i-1번째에서 고려했다.) 따라서 반복문 안에 있는 코드가 나왔다.
